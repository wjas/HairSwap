# 腾讯云 CloudBase 部署指南

## 🎯 CloudBase 架构方案

### 架构图

```
用户访问
    ↓
CloudBase 静态托管（前端）
    ↓
CloudBase 云函数（后端 API）
    ↓
├─ /generate - AI 生成
├─ /save-history - 保存历史记录
├─ /history-list - 获取历史记录列表
└─ /history/:id - 获取历史记录详情
    ↓
CloudBase 云数据库（存储历史记录元数据）
CloudBase 云存储（存储图片文件）
    ↓
火山引擎 API（AI 生成）
```

### 核心优势

| 特性 | 说明 |
|------|------|
| **免费额度** | 个人版免费额度：调用次数 20万次/月，容量 2GB，资源使用量 10万GBS，云函数外网出流量 2GB，CDN 5GB，回源流量 5GB |
| **小程序友好** | 完美支持微信小程序开发 |
| **一站式平台** | 静态托管、云函数、数据库、存储一体化 |
| **自动扩缩容** | 无需担心流量问题 |
| **微信生态** | 微信登录、支付、分享等深度集成 |
