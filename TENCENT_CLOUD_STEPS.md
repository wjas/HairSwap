# 腾讯云详细部署步骤

## 📋 前置准备

### 1. 注册腾讯云账号
- 访问：https://cloud.tencent.com/
- 注册账号并完成实名认证

### 2. 领取新用户福利
- 访问：腾讯云免费体验馆
- 领取新用户代金券（价值 2860 元）
- 领取免费试用产品（超过 60 款）

---

## 🔧 第一步：创建云函数 SCF

### 1. 进入云函数控制台
- 访问：https://console.cloud.tencent.com/scf
- 选择地域：推荐「广州」或「上海」

### 2. 创建函数
- 点击「新建」→「从头开始」
- 函数类型：选择「Web 函数」
- 函数名称：`hair-swap`
- 运行环境：Node.js 16
- 内存配置：128MB（或 256MB）
- 执行超时时间：60秒

### 3. 上传代码
- 我们会准备好 `tencent-cloud-function.zip` 代码包
- 选择「本地上传 ZIP 包」
- 上传准备好的代码包

---

## 🔐 第二步：配置环境变量

### 进入函数配置
- 点击函数名称 →「函数配置」→「编辑」

### 添加环境变量
| 变量名 | 值 | 说明 |
|--------|-----|------|
| `VOLCANO_API_KEY` | 你的火山引擎 API Key | 从火山引擎方舟获取 |
| `VOLCANO_MODEL` | `doubao-seedream-5-0-260128` | 模型名称 |
| `VOLCANO_ENDPOINT` | `https://ark.cn-beijing.volces.com/api/v3` | API 地址 |

---

## 🌐 第三步：配置 API 网关触发器

### 1. 创建触发器
- 点击「触发管理」→「创建触发器」
- 触发方式：选择「API 网关触发器」
- 触发服务：新建 API 服务
- 服务名称：`hair-swap-api`
- 发布环境：发布（release）

### 2. 获取访问地址
- 创建成功后，会显示「访问路径」
- 类似：`https://service-xxx.gz.apigw.tencentcs.com/release/`
- 这个就是你的 API 访问地址！

---

## 📦 第四步：准备部署代码包

### 代码包结构
```
tencent-cloud-function/
├── index.js          # 主入口文件
├── package.json      # 依赖配置
├── h5/               # 前端文件
│   ├── index.html
│   ├── js/
│   │   └── app.js
│   ├── styles/
│   │   └── main.css
│   ├── images/       # 发型模板（含缩略图）
│   └── history/      # 历史记录（可选）
└── node_modules/     # 依赖包（需要打包）
```

### 打包命令
```bash
cd tencent-cloud-function
npm install
zip -r ../tencent-cloud-function.zip .
```

## 🆘 常见问题

### Q: 云函数和 Web 函数的区别？
A: Web 函数更适合处理 HTTP 请求，我们的场景用 Web 函数更合适。

### Q: 历史记录存储在哪里？
A: 可以存储在云函数本地目录（临时），或者对象存储 COS（持久化）。

### Q: 费用如何计算？
A: Demo 阶段几乎免费，超出免费额度后按量计费，成本很低。
