# 腾讯云部署指南

## 🎯 腾讯云 Serverless 架构方案

### 架构图

```
用户访问
    ↓
API 网关（免费额度：100万次/月）
    ↓
云函数 SCF（免费额度：100万次/月 + 40万GB·s）
    ↓
├─ 处理 API 请求（生成、历史记录）
├─ 对象存储 COS（免费额度：50GB存储 + 6GB/月流量）
│  └─ 存储历史记录图片
└─ 火山引擎 API（AI 生成）
```

### 核心优势

| 产品 | 免费额度 | 用途 |
|------|---------|------|
| **云函数 SCF** | 100万次/月 + 40万GB·s | 运行后端代码 |
| **API 网关** | 100万次/月 + 1GB流量 | 提供 HTTP 访问 |
| **对象存储 COS** | 50GB标准存储 + 6GB/月流量 | 存储历史记录图片 |
| **云开发 CloudBase** | 一站式解决方案 | 可一键生成小程序后台 |

---

## 📋 部署步骤总览

1. 注册腾讯云账号
2. 领取新用户免费额度/代金券
3. 创建云函数 SCF
4. 配置环境变量
5. 配置 API 网关触发器
6. 配置对象存储 COS（可选）
7. 测试并部署
