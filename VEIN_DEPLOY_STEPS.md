## 📋 部署步骤（20 分钟）

### 步骤 1：登录火山引擎控制台（1 分钟）

1. 访问火山引擎控制台
   ```
   https://console.volcengine.com
   ```

2. 使用你的账号登录

---

### 步骤 2：进入函数服务（2 分钟）

1. **搜索函数服务**
   - 在顶部搜索框输入："函数服务" 或 "Function Graph"
   - 点击进入

2. **首次使用开通服务**（如果是第一次）
   - 点击"立即开通"
   - 同意服务条款
   - 点击"确认开通"

---

### 步骤 3：创建函数（5 分钟）

#### 3.1 点击"创建函数"
- 在函数服务页面，点击"创建函数"按钮

#### 3.2 选择创建方式
- 选择"从零开始创建"
- 点击"下一步"

#### 3.3 填写基本信息

```
┌─────────────────────────────────────┐
│ 函数名称：hairswap                  │
│ 描述：AI 换发型 Demo                 │
│ 运行环境：Node.js 18                │
│ 内存配置：512MB                     │
│ CPU 配置：1 Core                     │
│ 超时时间：60 秒                      │
│ 执行角色：默认即可                  │
└─────────────────────────────────────┘
```

#### 3.4 点击"下一步"

---

### 步骤 4：配置触发器（3 分钟）

#### 4.1 选择触发器类型
- 选择"API 网关触发器"
- 点击"创建新触发器"

#### 4.2 配置触发器

```
┌─────────────────────────────────────┐
│ 触发器名称：hairswap-trigger        │
│ 请求方法：ANY（支持所有方法）       │
│ 请求路径：/*                         │
│ 协议类型：HTTPS                     │
└─────────────────────────────────────┘
```

#### 4.3 高级配置
- 启用 CORS：勾选 ✅
- 允许来源：`*`

#### 4.4 点击"完成"

---

### 步骤 5：上传代码（3 分钟）

#### 5.1 进入代码管理
- 创建成功后，点击函数名称进入详情页
- 点击"代码"标签

#### 5.2 上传代码包
- 点击"上传代码包"
- 选择本地文件：`/Users/AS/codex-project/HairSwap/vein-function.zip`
- 等待上传完成

#### 5.3 确认入口文件
- 运行入口：`index.handler`
- 应该自动识别

#### 5.4 点击"保存并部署"

---

### 步骤 6：配置环境变量（2 分钟）

#### 6.1 进入配置页面
- 点击"配置"标签
- 找到"环境变量"部分

#### 6.2 添加环境变量

点击"添加环境变量"，依次添加 3 个：

**第 1 个：**
```
Key: VOLCANO_API_KEY
Value: bbb3102e-6d74-4607-823c-573fbe2e7f...（你的完整 API Key）
```

**第 2 个：**
```
Key: VOLCANO_MODEL
Value: doubao-seedream-5-0-260128
```

**第 3 个：**
```
Key: VOLCANO_ENDPOINT
Value: ng.volces.com/api/v3/images/generations
```

#### 6.3 点击"保存"

## ⚠️ 重要提示

### 1. 历史记录功能
当前版本简化处理，历史记录可能无法保存。如需完整功能，需要：
- 配置对象存储（TOS）
- 或配置数据库

### 2. 性能优化
如果生成速度慢，可以：
- 增加内存到 1024MB
- 增加超时时间到 120 秒

### 3. 成本控制
- 每月 100 万次免费调用
- 超出后约 0.0001 元/次
- 对于 Demo 完全够用

## 🔧 常见问题

### Q1: 500 错误怎么办？
A: 检查环境变量是否正确，API Key 是否有效

### Q2: 404 错误怎么办？
A: 检查触发器路径配置，确保是 `/*`

### Q3: CORS 错误怎么办？
A: 在触发器配置中启用 CORS，允许来源设为 `*`

### Q4: 超时错误怎么办？
A: 增加函数超时时间到 120 秒
