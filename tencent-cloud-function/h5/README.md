# HairSwap H5 Demo

## 快速开始

### 1. 准备发型模板

将发型模板图片复制到 `images` 目录：

```bash
cd /Users/AS/codex-project/HairSwap/h5
cp ../发型 01.png images/
cp ../发型 02.png images/
```

### 2. 本地预览（无需服务器）

直接打开 HTML 文件：

```bash
# Mac
open index.html

# 或者使用 Python 简单服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

### 3. 手机测试

**方案 A：同一 WiFi 网络**
```bash
# 查看本机 IP
ifconfig | grep "inet "

# 手机访问 http://你的 IP:8000
```

**方案 B：使用内网穿透工具**
- ngrok
- localtunnel
- frp

## 部署云函数

详见 [DEPLOYMENT.md](./DEPLOYMENT.md)

### 简要步骤：

1. 登录火山引擎控制台，创建云函数
2. 上传云函数代码（见 DEPLOYMENT.md 示例）
3. 配置环境变量 `VOLCENGINE_API_KEY`
4. 创建 API 网关触发器
5. 复制 API 地址到 `js/app.js` 的 `apiBaseUrl`

## 功能说明

### 核心功能

1. **上传照片**
   - 支持拍照或从相册选择
   - 自动压缩和格式转换
   - 实时预览

2. **选择发型**
   - 网格展示发型模板
   - 点击选择，视觉反馈
   - 支持扩展更多发型

3. **AI 生成**
   - 调用火山引擎 doubao-seedream-5-0-260128 模型
   - 显示加载动画和预计时间
   - 自动错误处理

4. **结果展示**
   - 高清展示生成效果
   - 支持保存图片到相册
   - 支持重新生成

### 技术特点

- **纯前端实现**：无需框架，轻量快速
- **移动端优先**：响应式设计，完美适配手机
- **无存储方案**：保护隐私，零成本
- **错误处理**：完善的提示和重试机制

## 常见问题

### Q: 图片无法上传
**A:** 检查：
- 图片格式是否为 PNG/JPG
- 文件大小是否超过 10MB
- 浏览器权限设置

### Q: 发型模板加载失败
**A:** 检查：
- 图片是否在 `images` 目录
- 文件名是否正确（区分大小写）
- 使用本地服务器而非 file:// 协议

### Q: 生成失败
**A:** 检查：
- 云函数是否部署成功
- API 地址是否正确
- API Key 是否有效
- 查看浏览器控制台错误信息

### Q: 保存失败
**A:** 
- iOS Safari：可能提示"无法下载此文件"，建议长按图片保存
- Android Chrome：应该可以直接下载

## 性能优化建议

1. **图片压缩**
   - 前端上传前压缩到合适大小
   - 建议宽度不超过 1024px

2. **CDN 加速**
   - 发型模板使用 CDN 托管
   - 云函数配置 CDN

3. **缓存策略**
   - 静态资源设置长缓存
   - 发型模板使用版本号
