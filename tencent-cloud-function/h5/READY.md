## 📱 使用步骤

1. **上传照片**
   - 点击"上传照片"区域
   - 选择照片或拍照
   - 确认预览正确

2. **选择发型**
   - 从 4 个发型中选择一个
   - 点击发型模板
   - 确认有✓标记

3. **生成效果**
   - 点击"立即生成"按钮
   - 等待 10-20 秒
   - 查看 AI 生成的效果

4. **保存结果**
   - 点击"保存图片"
   - 或长按图片保存

## 💡 常见问题

### 点击"立即生成"没反应？
1. 检查是否已上传照片
2. 检查是否已选择发型
3. 查看浏览器控制台错误（F12）

### 显示"本地服务未启动"？
确保 test-server.js 正在运行：
```bash
cd /Users/AS/codex-project/HairSwap/h5
node test-server.js
```

### 生成失败？
1. 检查 .env 文件中的 API Key 是否正确
2. 查看终端输出的错误信息
3. 确认网络连接正常
