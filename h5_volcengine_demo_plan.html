火山引擎发型生成 H5 Demo 工程方案（最小可行）

目标：让用户只需上传/拍照 → 点击生成 → 得到换发型后的照片。无需登录豆包网页。

一、关键结论（先看）

  • 可行：用火山引擎 doubao-seedream 5.0 lite 多图融合模型实现“参考图发型替换”，H5 只负责上传和展示。
  • 最简单做法：前端 H5 + 云函数（后端代理 API Key）+ 云存储。
  • 模型能力：doubao-seedream 5.0 lite 支持多图输入（2-14张）+文本提示词生成图片，可实现客户图 + 发型模板图的融合。

二、产品边界与 MVP 范围

用户侧体验

  1. 选择/拍摄客户照片（正面）。
  2. 选择一个发型模板（可选；也可先固定一个模板）。
  3. 点击“生成”，等待 10–30 秒。
  4. 看到结果图，可保存到相册。

MVP 只做这些

  • 仅支持正面角度。
  • 模板发型数量 1–3 个。
  • 无需登录、无需填写信息。

三、整体架构（最简单）

  • H5 前端：上传图片、选择模板、展示结果。
  • 云函数：保存 API Key，调用火山引擎图像编辑 API。
  • 云存储：存放用户上传图和结果图（若 API 直接返回 URL，也可不存）。

四、关键功能拆解

功能 最简实现 备注
上传客户图 H5 直接上传到云存储 可限制大小，减少等待
选择模板 前端固定模板列表 后期可扩展
生成发型 云函数调用火山 API 只改头发，保留脸部/背景
展示结果 前端展示返回图 URL 提供“保存”按钮

五、火山引擎 API 使用策略

模型选择：doubao-seedream 5.0 lite，支持多图融合，可实现客户图 + 发型模板图的发型替换。

已确认能力

  1. API：POST https://ark.cn-beijing.volces.com/api/v3/images/generations
  2. 支持多图输入（2-14张）+ 文本提示词生成单张图片
  3. 图片支持：客户图 + 发型模板图 + 提示词 = 换发型后图片
  4. 返回结果形式：base64 / URL（待实际调用确认

API 使用方式

  • 输入：客户图（URL或Base64） + 发型模板图（URL或Base64） + 提示词。
  • 提示词示例： 
    请保持脸部、五官、背景不变，仅将发型替换为参考图中的发型，发际线自然，发色一致。
  • 推荐尺寸：2048x2048（默认），总像素范围 3,686,400 - 16,777,216

模型参数

  • model: doubao-seedream-5.0-lite
  • sequential_image_generation: disabled（生成单图模式）
  • image: [客户图URL, 发型模板图URL]
  • size: 2048x2048（或2K/4K）

六、工程落地方案（最简单路线）

1) 选择托管（推荐）

  • 前端：免费静态托管（如腾讯云 CloudBase、阿里云 OSS 静态网站）。
  • 后端：云函数（与托管同平台，省配置）。

2) 目录结构（示例）

/project
  /web           # H5 前端
    index.html
    app.js
    style.css
  /server        # 云函数
    index.js
  /assets        # 模板发型图（小量）
  config.example.json

3) 前端交互（最小）

  • 上传控件（拍照/相册）。
  • 模板选择（1–3 张缩略图）。
  • 按钮：生成。
  • 结果图：生成后显示，提供“长按保存”提示。

4) 后端云函数职责

  • 接收前端请求（客户图 URL + 模板图 URL）。
  • 拼接提示词，调用火山引擎 API。
  • 返回结果图 URL 或 base64。

七、数据与安全

  • API Key 只放云函数，前端不暴露。
  • 图片仅用于生成，定期清理（如 7 天）。
  • 提示用户“照片将用于 AI 处理”。

八、测试标准（是否可行）

  • 头发替换成功，脸部、五官、背景基本不变。
  • 发际线自然，没有明显糊边/拉伸。
  • 生成时间可接受（10–30 秒）。

九、成本预估（粗略）

  • 云函数与静态托管：免费额度足够 demo。
  • 火山引擎 API：按调用计费，小量测试成本低。

十、实施步骤清单（按顺序）

  1. 注册火山引擎账号，开通 doubao-seedream-5.0-lite 模型，获取 API Key。
  2. 准备 1–3 张模板发型图（正面，清晰）。
  3. 开通云函数与静态托管。
  4. 写一个最简单的 H5 页面（上传 + 生成 + 展示）。
  5. 云函数调用火山 API，返回结果。
  6. 手机访问 H5 测试效果。
