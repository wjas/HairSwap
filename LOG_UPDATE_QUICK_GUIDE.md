#  更新迭代日志快速指南

## ⚠️ 最重要的事

**加时间！加时间！加时间！**

```
❌ 错误：2026-03-10 修复 bug
✅ 正确：2026-03-10 16:45 修复 bug
```

## 🚀 三步更新法

1. **打开文件**：`iteration-log.html`
2. **看顶部注释**：有详细格式和检查清单
3. **复制旧日志格式**：照着已有日志的样子写

## ✅ 快速检查

- [ ] 时间加了吗？（HH:MM）
- [ ] 版本号更新了吗？
- [ ] 原因和方案写了吗？

## 📋 完整格式

```html
<article>
  <div class="bar">
    <span>2026-03-10 16:45 日志标题</span>
    <span>v1.5.x</span>
  </div>
  <div class="content">
    <h2>修复</h2>
    <ul>
      <li>🐛 修复内容...</li>
    </ul>
  </div>
</article>
```

---
**记住：每次更新都要加时间！**
